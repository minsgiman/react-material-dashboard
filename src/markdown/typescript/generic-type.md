# 제네릭(Generic) 타입

### Type Assertion

* C# 및 Java와 같은 언어에서 재사용 가능한 컴포넌트를 만들기 위한 주요 도구 중 하나는 제네릭 즉 단일 타입이 아닌 다양한 타입을 처리할 수 있는 컴포넌트를 만드는 것입니다.
